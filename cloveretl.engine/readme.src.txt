CloverETL engine version 2.1.0
------------------------------

Use Apache ANT to build binary - run with
"build.xml" from "cloveretl.engine/" dir. 

Before compilation, update "cloveretl.engine/build.properties" file
to reflect your set-up.

Requires Java version 1.5. All other third party
libraries required can be found in "cloveretl.engine/lib/" directory.

These are:
commons-logging - http://jakarta.apache.org/commons/logging/
log4j 		- http://logging.apache.org/log4j/docs/
poi		- http://jakarta.apache.org/poi/
jxl		- http://jexcelapi.sourceforge.net/
jms		- http://java.sun.com/products/jms/
javolution	- http://www.javolution.org/
commons-cli	- http://jakarta.apache.org/commons/cli/

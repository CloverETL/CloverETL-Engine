CTL2FunctionsTutorial

by Tomas Waller (06/18/2010)


This project aims to present an overview of built-in CloverETL functions.

All of these graphs consist of UniversalDataReader (or DataGenerator), Reformat, and UniversalDataWriter.

Reader reads (or DataGenerator generates) data records that are used as argument values for corresponding function.

Transformation defined directly in Reformat is always written in new version of Clover Transformation Language (CTL2).

CTL functions of this CTL2 are used in all Reformat-s.

Writer writes the overview of the whole function:

  • Input arguments
  • Function pattern
  • Function output

Each graph bears a note describing corresponding function.

Since version 3.0 of CloverETL, this transformation language can be used to define transformations in addition to the older CTL1. (Transformations can also be written in Java.)

Unlike CTL1, this new language is strongly typed and we suggest you use it instead of CTL1.

For better understanding of CTL2, see CTL2 description.

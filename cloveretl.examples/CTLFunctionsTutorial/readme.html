CTL1FunctionsTutorial

by Tomas Waller (06/18/2010)


This project aims to present an overview of built-in CloverETL functions.

All of these graphs consist of UniversalDataReader (or DataGenerator), Reformat, and UniversalDataWriter.

Reader reads (or DataGenerator generates) data records that are used as argument values for corresponding function.

Transformation defined directly in Reformat is always written in older version of Clover Transformation Language (CTL1).

CTL functions of this CTL1 are used in all Reformat-s.

Writer writes the overview of the whole function:

  • Input arguments
  • Function pattern
  • Function output

Each graph bears a note describing corresponding function.

Before version 3.0 of CloverETL, this Clover Transformation Language (CTL1) was used to define transformations. (They can also be written in Java.)

Since version 3.0 of CloverETL, a new transformation language appears - it is the strongly typed CTL2. See the CTL2FunctionsTutorial project for the overview of functions of CTL2.

However, this CTL1 can still be used. For better understanding of CTL1, see CTL1 description.

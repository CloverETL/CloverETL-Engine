How to run jobflow examples

Jobflows require CloverETL Server

Jobflows are CloverETL workflow descriptions that run only in the CloverETL Server environment.

Based on how you evaluate the product, you can either use your own trial instance of the CloverETL Server or use the publicly available CloverETL Server Demo.

Using the public demo

You can test drive all the examples on our publicly available demo Server instance. A copy of these examples is already present on the demo - just connect to the demo Server from within your Designer and you're ready to go.

CloverETL Server URL for Demo:
http://server-demo-ec2.cloveretl.com/clover/
Username: clover
Password: clover

Your own CloverETL Server trial instance

If you got your own trial copy of the CloverETL Server, you can test the jobflow examples right away as the standard demo installation comes bundled with all the examples. Just navigate to one of the Examples sandboxes, or use the Designer and connect to your local Server installation.

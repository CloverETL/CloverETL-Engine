Running CloverETL examples

by Pavel Najvar, Last modified

Detailed examples description.

Using examples in CloverETL Designer

CloverETL Designer contains all examples built-in. You can import them into your workspace using File->New->Other, CloverETL->CloverETL Examples project. For more information visit CloverETL Examples Project.

Installing and running examples in CloverETL Engine

  • Download a binary package of CloverETL engine and Clover Examples or build one from sources.
  • Extract CloverETL engine and Clover Examples to the same directory
  • Ensure that cloverETL/bin/clover (clover.bat) is executable.
  • Go to project directory you want to run graph from (e.g cloverETL/examples/simpleExamples)
  • Use clover (clover.bat) script to run a graph, eg.:
    ../../bin/clover graph/graphPartition.grf

Important notes

  • CloverETL requires Java (JVM) version 1.6 or higher!
  • Database examples in ExtExamples use Derby database - please follow enclosed ExtExamples/README.txt
  • JavaExamples project contains examples, that illustrate how to use CloverETL from java
